What is Se7en?

Se7en is a best-of-breed administration template. We employ stunning design, the latest technologies and industry best practices to create a one of kind administration template suitable for many needs.

Features
  • Clean and sleek design
  • 100% Mobile ready / responsive layout
  • Valid HTML5 and CSS3
  • Cross-browser compatible (Chrome/Safari/Firefox/IE8+)
  • SASS and Coffeescript ready
  • Built with Bootstrap 3
  • Clean developer friendly markup
  • Image gallery
  • Retina ready
  • Custom buttons and colors
  • 5 color themes to choose from
